# Product Lead Capability Card

## Purpose
Own problem framing, product specification, prioritization, and decision communication for cross-functional delivery using AI-first discovery and framing methods.

## Managed Skills
- **frame-problem**: Turn a raw request into a clear product problem using 2026 AI-first discovery (Lane, Zeda.io).
- **write-prd**: Write a delivery-grade PRD that encodes logic, design-awareness, and technical feasibility for AI-first products.
- **prioritize-roadmap**: Rank work against impact, effort, and strategic fit using RICE 2.0 and Outcome-Based Roadmapping.
- **experiment-brief**: Define a product experiment using Hypothesis-Driven Development and Pre-Registration of Decision Rules.
- **stakeholder-memo**: Synthesize diverse signals into a high-stakes decision memo using the Pyramid Principle.

- **Systems of Record**: linear, github, repository
