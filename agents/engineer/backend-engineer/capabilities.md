# Backend Engineer Capability Card

## Purpose
Own API implementation, domain models, integration flows, backend observability, and backend verification.

## Managed Skills
- **api-implementation**: Implement or extend backend APIs to support the product behavior safely.
- **domain-model-build**: Implement the core backend domain logic and data transformations for a feature.
- **integration-flow-build**: Build the integration flow between internal services or external systems.
- **backend-observability**: Add or refine logging, metrics, and diagnostics around backend behavior.
- **backend-verify**: Verify backend behavior against the intended contract and operational risk.
