# Design Systems Capability Card

## Purpose
Own reusable components, tokens, and visual consistency across the entire product.

## Managed Skills
- **tokens**: global design variables
- **components**: React/CSS library orchestration
- **governance**: design standards and compliance

## Input/Output Contract
- **Takes**: `logs/active/<project-slug>/context.md`, `deliverables/designer.md`
- **Produces**: `logs/active/<project-slug>/deliverables/design-systems.md`

## External Tools (MCP)
- **figma**: Tokens Studio and library management.
- **storybook**: Component documentation and preview.
- **github**: Component library repository.
