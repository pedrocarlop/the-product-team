# Content Designer Capability Card

## Purpose
Own microcopy systems, state messaging, naming, guidance, localization preparation, and product content reviews.

## Managed Skills
- **microcopy-flow-design**: Write the core product copy across a flow so users know what is happening and what to do next.
- **error-empty-success-states**: Design the messaging for non-happy-path states so users can recover or proceed confidently.
- **naming-and-taxonomy**: Define names and labels so concepts, navigation, and settings stay coherent.
- **conversation-and-guidance-design**: Write guided help, assistant, or in-product guidance content that supports task completion.
- **localization-prep**: Prepare content for translation, expansion, and locale-sensitive adaptation.
- **content-review**: Review an existing content surface for clarity, consistency, and voice.
