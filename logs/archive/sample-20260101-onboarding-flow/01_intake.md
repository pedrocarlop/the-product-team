# Intake

**Objective**: Build a guided onboarding flow that walks new employees through profile setup, team assignment, tool access requests, and a welcome checklist.

**Constraints**:
- Must work on mobile and desktop
- Must integrate with existing auth system
- Must complete within current sprint (2 weeks)

**Dependencies**:
- HR team provides the onboarding checklist content
- Backend API for employee profile already exists

**Ambiguities and risks**:
- Unclear whether tool access requests need manager approval flow
- Mobile breakpoints not defined in current design system

**Initial staffing hypothesis**: Product Designer + Frontend Engineer + UX Flow Reviewer
