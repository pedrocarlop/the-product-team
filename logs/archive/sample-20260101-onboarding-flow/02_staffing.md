# Staffing

## Selected Roles

| Role | Fit-Check | Outcome |
|---|---|---|
| product-designer | Accepted | Full ownership of task flows, screen design, and component mapping |
| frontend-engineer | Accepted | Full ownership of implementation and design-to-code translation |
| ux-flow-reviewer | Accepted | Will review after implementation is complete |

## Declined Roles

| Role | Rationale | Recommended Replacement |
|---|---|---|
| backend-engineer | Declined — existing API covers all needed endpoints, no backend changes required | None needed |

## Final Ownership Map

- **product-designer**: Task flows, screen states, component specs, interaction patterns
- **frontend-engineer**: Component implementation, state management, responsive behavior
- **ux-flow-reviewer**: Post-implementation flow validation
