# Routing Decision

**Request**: Design and implement a new employee onboarding flow for the HR portal.

**Complexity**: Multi-step feature involving UX design, frontend implementation, and review.

**Best possible team assessment**: This requires a product designer for UX and flows, a frontend engineer for implementation, and a UX flow reviewer to validate. A product manager is not needed — requirements are already clear from the HR team's brief.

**Decision**: Orchestrated workflow.

**Rationale**: Cross-functional ownership (design + engineering), meaningful specialist judgment needed, review gate warranted before shipping.
