# Scenario: Engineering - High-Fidelity Frontend Component

## Task
Implement a 'Project Health Card' component using React and Tailwind CSS.

## Requirements
1. **Framework**: React (functional component).
2. **Styling**: Tailwind CSS.
3. **Icons**: Use `lucide-react`.
4. **Glassmorphism**: The container must have a translucent background (e.g., `bg-white/10`) and `backdrop-blur-md`.
5. **Gradients**: Use a subtle linear gradient for the progress bar or status badge.
6. **Responsiveness**: Ensure the card is responsive and uses appropriate padding/spacing.

## Output Contract
Produce a `component.tsx` file (or a markdown block containing the code) that includes the full implementation.
